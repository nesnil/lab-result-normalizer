# 骨科出院小结标准章节模板

阶段 3（章节组装）查本文件。定义出院小结的章节顺序、每章内容要求、必填/选填、本院固定套句，以及完整性校验所需的必填项清单。

> **本模板已依真实案例校准**：结构、字段名、固定套句均提炼自真实骨科出院记录（南京医科大学附属常州第二人民医院两例：踝关节骨折内固定、人工膝关节单髁置换），标注【实例校准】。这保证 skill 输出的格式与真实病历一致、可直接供医生审核。医生如需适配其他医院模板，改本文件对应行即可。

> 🔴 **忠实性红线（本模板所有"固定套句"仅是句式骨架，不是可填充的默认内容）**：套句里的每个具体值（评分、药名、日期、次数、部位）**必须来自原始材料原文**；**原文没有的段落/事件/医嘱一律不写**（如 MDT、健康宣教、分阶段康复方案默认不出现）。**床号**取出院时最终值（不写"75→50"这类过程值，只写 50）；**入院/出院日期不带时分**（写 2026-06-25，不写 2026-06-25 10:00）。详见 SKILL.md「忠实性红线」。

## 二、章节结构与内容要求

| 序号 | 章节 | 必填 | 内容要求 | 主要来源 |
|---|---|---|---|---|
| 1 | 页眉基本信息 | 是 | 姓名、性别、科别、床号、住院号 | 病案首页 / 入院记录页眉 |
| 2 | 人口学信息 | 是 | 姓名、性别、年龄、婚姻、职业 | 病案首页 / 入院记录 |
| 3 | 入院诊断 + 入院日期 | 是 | 诊断按"1. 2."编号，骨科主诊断在前（含部位+侧别+分型），合并症在后 | 入院记录"入院诊断" |
| 4 | 手术名称 + 手术日期 | 手术者是 | 规范术式名（部位+术式+入路+内固定），带序号；非手术患者省略此行 | 手术记录 / 病案首页手术表 |
| 5 | 出院诊断 + 出院日期 | 是 | 通常同入院诊断；如病程有修正需与病程一致 | 病案首页出院诊断 / 术后诊断 |
| 6 | 入院情况 | 是 | 见 §三·1（主诉+既往要点+查体+专科+入院辅检，压缩提炼） | 入院记录（压缩） |
| 7 | 诊疗经过 | 是 | 见 §三·2（叙事式一大段：入院处置→完善检查→手术→术后恢复复查→出院决策） | 病程记录+手术记录+检验检查 |
| 8 | 出院情况 | 是 | 见 §三·3（出院转归+伤口愈合固定字段 + 出院病情情况） | 病程记录（出院前查体） |
| 9 | 出院医嘱 | 是 | 见 §三·4（只写原文医嘱；休息营养、换药、负重康复、复诊、不适随诊等原文有才写） | 原始出院记录 / 医嘱单 / 病程末次查房 |
| 10 | 随访区块 | 是 | 复查日期、复查内容、是否推荐他科、X光/CT/MRI/病理号、签收/签名 | 检查系统 / 医嘱 |
| 11 | 免责声明 | 是 | AI 辅助生成草稿，需经治医生审核签字（附于文末） | 本 skill 固定 |

## 三、各章节写法与本院固定套句【实例校准】

### 1. 入院情况（压缩提炼，非全文搬运）
- **首句固定句式**：`患者因"【主诉原文】"入院。`
- 既往史：仅保留关键项（如高血压病史+用药情况），源自入院记录既往史，其余（否认肝炎史等）可略。
- 查体：生命体征（体温/脉搏/呼吸/血压）+ 关键阳性体征；全身查体阴性项可高度压缩。
- **专科情况**：**整段抽取**专科查体，不得只截取患肢部分——含脊柱/骨盆等阴性描述句（如"颈椎、胸椎、腰椎生理弯曲存在""骨盆分离挤压征阴性"）+ 患肢阳性体征，逐句保留，不漏抽。短句不得同义改写，例如原文写"颈椎、胸椎、腰椎生理弯曲存在"时，不得概括成"脊柱无明显畸形"；原文有"生理反射存在，病理反射未引出"时不得省略。
- 辅检：入院时影像**照原文抄录**（检查名+科室+日期+时间+结论），**不改侧别、不删科室/时间戳**。
  - 🔴 **侧别等矛盾禁止静默"纠正"**：若检查名称侧别与结论/诊断侧别不一致（如原文"右侧膝关节…片：左侧膝关节退行性变"），**正文保留原文原样**，同时在**质控清单**列"侧别矛盾（检查名 vs 结论/诊断）待医生核实"。**绝不可把'右侧'悄悄改成'左侧'来'消除'矛盾**——发现并暴露矛盾正是本 skill 的职责。

### 2. 诊疗经过（叙事式一大段，含本院固定套句）
按顺序拼接：
- **入院处置套句**（创伤示例）：`患者入院后予以骨科护理常规，一级护理，普食，患侧肢体制动，预防深静脉血栓形成，消肿，止痛，改善微循环，预防感染等对症治疗；予完善相关检验检查，评估病情排除禁忌后择日手术。`（关节择期手术可简化为"入院后完善相关检查"）
- **完善检查**：`完善检查示：` + **逐条逐字罗列**检验检查结果（检查名+科室+日期+结果原文）。
  - 🔴 **禁止概括为"未见明显异常"等摘要句**：哪怕原文多数正常，也须逐条列出；**凡异常结果（血沉↑、心超"左房增大伴二尖瓣反流"、脂肪肝、D-二聚体↑、CRP、WBC↑等）一个都不能省、不能被"未见异常"盖掉**。
  - 每个数值、单位、日期、科室、时间戳**逐字取自原文**，不改写、不近似、不摘要。
  - （真实案例诊疗经过确实不必把每张化验单每个指标全抄，但**取舍的边界是"异常必留、数值不改"**，绝不能把有异常的检查写成"未见异常"——这正是本 skill 要抓的质控点，不可自己犯。）
- **长病程记录全程覆盖**：完善检查和术后复查必须从病程记录全量时间线抽取，尤其不能漏掉手术后至出院日的后半段。出院日前复查的凝血、D-二聚体、CRP/血沉、术后 X 线、下肢血管超声等，若源文有，必须进入诊疗经过或质控清单。
- **VTE 评分/预防**（若原文病程"入院评估"有）：Caprini 评分**几分写几分**（原文几分照写）；**如入院与术后为不同分值（如入院 3 分、术后 10 分），须区分时点或注明以术后为准**，不得只取其一而不说明。抗凝**药名逐字照原文**，不得替换同类药。原文没有的评分/药名不编。
- **手术**：`未见明显手术禁忌，于YYYY-MM-DD在【麻醉方式】下行【规范术式】，术中所见：【术中所见原文摘录】。`⚠️ **原文没有 MDT/术前讨论/会诊就绝不写**；不得因"骨科手术常有 MDT"而编造。手术入路按 raw/std 双轨：正文保留手术记录原文（切口、解剖间隙、入路原句），规范化入路只可括注或进结构化字段；原文未写"外侧入路"时，不得推断写成"术中以腓骨外侧入路"。
- **术后处置+恢复套句**：`予【吸氧、预防感染、镇痛、防止静脉血栓、消肿、改善微循环等】对症支持治疗，并注意生命体征变化，患肢感觉、运动及末梢血运变化。鼓励并指导患者适当功能锻炼。患者术后恢复可，` + **术后复查结果（逐条逐字，含影像结论如"假体位置良好/内固定后改变"+复查凝血/炎症指标+日期）；术后复查的异常值同样不得遗漏或概括**。括号内处置项目只写源文有的，原文有"予吸氧"必须保留，原文无则不编。
- **敏感事件（抢救/病危/严重并发症/输血）**：抽到抢救记录、病危、伤口红肿渗液+静脉抗生素等严重并发症时，**不直接在正文展开叙事**（真实医生版常刻意不写入交患者的文书）；改为在正文极简一句或不写，并在**质控清单**提示"病程记录 X 日有【抢救/并发症】记录，是否写入出院记录请医生决定详略"。
- **出院决策套句**：`患者目前病情平稳，伤口无红肿渗出及坏死，可予出院休养，予出院。`

### 3. 出院情况
- 固定字段行：`出院转归：好转` ｜ `伤口愈合：I/未拆线`（切口甲级、未拆线；已拆线则相应改）。
- **出院病情情况**：首选病程记录出院日/末次查房记录。若其中已有全身情况、体格检查、患肢切口与末梢血运（如"神志清楚，双瞳等大等圆，对光反射灵敏……患肢伤口敷料干燥，无渗血渗液，患肢活动受限，末梢血运感觉正常"），必须直接写入，不得标"未见原始记录"。常见三段为：①全身情况（"无头晕头痛…食纳睡眠可，大小便正常"）②体格检查（神志、瞳孔、心肺、腹部）③患肢（"伤口敷料干燥，无渗血渗液，末梢血运感觉正常"）。

### 4. 出院医嘱（**忠实原文，只写原始材料里已有的医嘱；原文几句就几句**）
> ⚠️ 医嘱是**忠实性重灾区**，严禁把知识库的分阶段康复/VTE/健康宣教灌进来（真实案例出院医嘱常仅 1–5 句）。以原始出院记录/医嘱单原文为准；若这些未提供，则病程记录出院日/末次查房中明确写有"出院医嘱："的段落就是合法首选来源，必须忠实照写/归纳，不能降级为"参考"。真实案例即为：
> - 案例1（踝）："注意休息，营养，注意伤口卫生，门诊定期换药，患肢避免完全负重，指导功能锻炼，周四下午黄建峰副主任医师、周四上午苗凯松医师门诊复查。不适随诊。"
> - 案例2（膝）："1、使用助行器行下肢功能锻炼。2、门诊复查伤口情况，视情况决定是否换药，术后三周拆线。3、一周后门诊复查（…医师门诊）。"

出院医嘱常见要素（**仅当原文有时才写**，无则略，不臆造）：
- 休息、营养、伤口卫生（原文若有）。
- 换药 / 拆线时间（原文若明确，如"术后三周拆线"；原文未写则不编）。
- 负重与功能锻炼：**照写原文措辞**（如"患肢避免完全负重，指导功能锻炼"）。**不得**自行扩写为"术后0–2周/2–6周…"分阶段方案——那属知识库通用建议，进质控 💡「可选建议」，不进正文（除非用户明确要求）。
- 门诊复诊（原文有则照写医生+时间；无则【待补充：复诊医生及时间】）。
- 不适随诊（原文若有）。
- **用药**：仅列原文出院医嘱中明确的药物；原文医嘱段无药物列表则**不列**（不得从术后医嘱/常识推断补药，如塞来昔布等）。

### 5. 随访区块字段
- 院外随访复查日期（病程末次查房有则照写，无则【待补充】）、复查内容（有则照写，无则【待补充】）、是否推荐他科（只有源文明确"否/是"才填写；无依据则【待补充】，不得默认"否"）、X光片号/CT号/MRI号/病理检验号（源自检查报告，无则留空或【待补充】）、签收人/医师签名。

## 四、必填项清单（阶段 5 完整性校验用）

以下任一缺失，进质控清单"缺失必填项"：
- 患者姓名、性别、年龄、住院号、科别
- 入院日期、出院日期
- 入院诊断（主诊断）、出院诊断（主诊断）
- 主诉 / 现病史摘要
- 诊疗经过
- 出院情况（出院转归 + 出院病情情况）
- 出院医嘱（负重/功能锻炼 + 复诊/随访 至少各一项）
- 手术患者：手术日期 + 规范术式名称 + 术中所见
- 随访区块：院外随访复查日期或内容
- 文末固定要素：门诊病历交付句、签收人、医师签名

## 五、逻辑校验规则（阶段 5 一致性校验用）

| 规则ID | 校验 | 触发提示 |
|---|---|---|
| D1 | 出院日期 ≥ 入院日期 | "出院日期早于入院日期，请核对" |
| D2 | 住院天数 = 出院 − 入院（按本院口径） | "住院天数与起止日期不符" |
| D3 | 手术日期 ∈ [入院, 出院] | "手术日期不在住院期间，请核对" |
| D4 | 出院诊断部位/侧别 与 手术术式部位/侧别 一致 | "出院诊断与手术术式的部位/侧别不一致" |
| D5 | 患肢侧别（左/右）在各资料间一致（含**检查名称侧别 vs 结论/诊断侧别**）。🔴 发现矛盾时**正文保留原文、进质控**，**严禁静默改写侧别去"消除"矛盾** | "侧别矛盾（检查名/结论/诊断之间）待医生核实" |
| D6 | 出院诊断 与 入院诊断 的演变合理（有修正应在病程体现） | "出院诊断较入院诊断有变化但病程未见说明" |
| D7 | 术式名部位 与 术中所见部位 一致 | "术式名称与术中所见部位不一致" |
| D8 | 诊疗经过引用的检验/影像日期 ∈ [入院, 出院] | "诊疗经过引用的检查日期不在住院期间" |
| D9 | 检验检查结果**未被概括为"未见明显异常"而丢弃异常值**；异常结果（血沉↑/心超异常/D-二聚体↑/CRP/WBC↑等）均已保留 | "检验结果被摘要化，异常值可能丢失，请逐条核对" |
| D10 | 数值前无"约/大约"等模糊词（源文为确切值时逐字照写） | "数值前出现模糊词'约'，应逐字照原文" |
| D11 | 长病程记录末段已覆盖；最后一条/出院日查房中的出院医嘱、出院查体、院外随访、术后复查已抽取 | "病程记录末段可能漏抽；源文有出院日信息却被标待补充" |
| D12 | OCR 噪声与多来源冲突已处理；疑似箭头误并入数值、S/CD vs S/CO 等需标质控 | "疑似 OCR 噪声/来源冲突，请核对原文" |
| D13 | 查体、神经反射、术后处置、手术入路等未被同义改写或推断成事实 | "原文短句被改写/省略，或规范化结果替代了原文事实" |

> 待医生补充：本院出院小结抬头、坐诊复诊安排、是否纳入专科评分（如 VTE-Caprini 评分、疼痛 VAS 评分——真实病程记录中已有，可选纳入体现质控特色）。
